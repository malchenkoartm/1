# Субпиксельный Анализ Сдвига

## Установка

```bash
pip install numpy opencv-python scikit-image scipy matplotlib
```

## Быстрый старт

1. Запустите приложение
2. Загрузите два изображения (кнопки "Изображение 1" и "Изображение 2")
3. Настройте ROI (координаты центра + размер, минимум 32x32)
4. Нажмите "ИЗМЕРИТЬ"
5. Результат отобразится в левой панели и на изображении

```bash
python main.py
```

Для запуска без перехода на верисю 3.10

```bash
python -3.10 main.py
```

## Настройки

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| Upsample | 200 | Точность (100-500) |
| Sigma | 0.8 | Сглаживание шума (0.5-1.5) |
| Min Conf | 0.5 | Порог достоверности |
| Размер ROI | 32 | Область измерения (мин. 32) |

## Результаты

```
ДВИЖЕНИЕ ОБЪЕКТА:
→ X = +2.47 px    (сдвиг по горизонтали)
→ Y = -1.33 px    (сдвиг по вертикали)
|Δ| = 2.81 px     (величина сдвига)
θ = -28.32°       (направление)
Достоверность: 0.92
```

## Навигация

| Действие | Управление |
|----------|------------|
| Зум | Ползунок на панели настроек |
| Панорама | Правая кнопка + перетаскивание |
| Сброс зума | Кнопка "1:1" |

## Сохранение

Кнопка "Сохранить результаты" создаёт в папке `results/`:
- `results_*.csv` — таблица
- `results_*.json` — данные
- `visualization_*.png` — изображение

## Требования

- Python 3.10
- Разрешение экрана: 1920x1080

## Алгоритм

Фазовая корреляция с апсемплингом (Guizar-Sicairos et al., 2008). Точность зависит от качества текстуры в ROI.
